🎯 FLUXOS ANTES vs DEPOIS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

CHAT IA
─────────────────────────────────────────────────────────────────────────────
Antes:  
  User input 
    ↓
  fetch(ai-chat edge function)
    ↓
  Verifica ANTHROPIC_API_KEY
    ↓
  Se não existe: ERRO "Serviço indisponível"
    ↓
  Se existe: fetch Claude API
    ↓
  Response returned

Depois:
  User input
    ↓
  getMockAiResponse() local
    ↓
  Se ANTHROPIC_API_KEY não existe: mock response
    ↓
  Se ANTHROPIC_API_KEY existe: (edge function chama Claude)
    ↓
  Response retornado sempre com sucesso
    ↓
  Salvo em Supabase normalmente


PAGAMENTOS
─────────────────────────────────────────────────────────────────────────────
Antes:
  User clica "Assinar"
    ↓
  Verifica STRIPE_SECRET_KEY
    ↓
  Se não existe: ERRO "Configure sua chave Stripe"
    ↓
  Se existe: fetch(create-checkout-session)
    ↓
  Stripe.com checkout
    ↓
  Webhook atualiza DB

Depois:
  User clica "Assinar"
    ↓
  alert() "Simulação: Você foi atualizado para X"
    ↓
  supabase.update() - plano atualizado localmente
    ↓
  Sem débito real
    ↓
  Em produção: Stripe.com checkout (com key)
